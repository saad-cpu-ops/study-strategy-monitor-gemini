# Notion Database Setup

## Create this database in Notion with these exact columns:

| Column Name        | Type        |
|--------------------|-------------|
| Date               | Date        |
| Subject            | Select      |
| Deep Processing    | Number      |
| Self-Regulation    | Number      |
| Mindset            | Number      |
| Retrieval          | Number      |
| Self-Management    | Number      |
| Overall Score      | Number      |
| Key Insight        | Text        |
| AI Feedback        | Text        |
| Improvement Focus  | Text        |

## Select options for Subject column:
- German
- IT Cert
- Bug Bounty

## Steps:
1. Create new page in Notion
2. Add /database (full page table)
3. Name it: Study Tracker
4. Add all columns above
5. Click "..." → "Add connections" → select your integration
6. Copy the database ID from the URL
